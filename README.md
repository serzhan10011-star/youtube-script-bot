# YouTube Script Bot

Telegram бот для генерации сценариев YouTube видео на основе тематики Reddit.

## Стек
- Python
- Telegram Bot API  
- Claude API (Anthropic)

## Функционал
- Генерация сценариев по заданной теме
- Адаптация под Reddit-стиль контента
- Поддержка разных форматов видео

## Установка
1. Клонировать репозиторий
2. Создать .env файл с ключами
3. pip install -r requirements.txt
4. python bot.py
